# Looping Examples

### Wrong Way ❌
```python
colors = ['black', 'pink', 'yellow', 'orange']

for i in range(len(colors)):
    print(colors[i])
```

### Better Way ✅ 
```python
colors = ['black', 'pink', 'yellow', 'orange']

for color in colors:
    print(color)
```
