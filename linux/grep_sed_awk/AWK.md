# AWK

### Built-In Variables

| Variable      | TBD                  |
| :---:         |     :---:      |
| FS   (" ")    | field seperator      |
| RS   ("\n")   | record seperator     |
| IFS  ("\n")   | delimeter            |
| OFS  (" ")    | output FS            |
| ORS  ("\n")   | output RS            |
| NR            | # of records         |
| NF            | # of fields          |
| SUBSEP        | null                 |
| IGNORECASE    | null                 |
| FILENAME      | return awk filename  |
