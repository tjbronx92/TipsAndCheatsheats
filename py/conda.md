
# Conda Commands

![image](/img/gi-joe-cobra-commander.gif "Cobra Commander")

## Conda Refresher 

### Update conda base
```bash
conda update -n base conda -y
```

### Create New venv
```bash
conda create -n [ENV_NAME] python=3.10 -y
```

### Activate venv
```bash
conda activate [ENV_NAME]
```

### List venvs
```bash
conda env list
```
